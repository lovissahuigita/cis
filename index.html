Biometric Attendance System

Software and hardware system to track users' attendance and report it to the owner. The attendance system uses biometric features to authenticate the users scanned to the system (MBED GT115C3 Fingerprint Scanner). The system is written in C. It compiles all authenticated users and reports it to the owner of the user group.

GT ICN Ticketing System

Database system that is implemented in MySQL, PHP, HTML, and CSS to record the information of the attendees of the Indonesian Cultural Night Event. The system keeps track of the tickets bought for the event and the prices for the attendees based on their demographics. The system is a closed system that can only be managed by specified administrators.

Yallpay.me

Web-application service that allows users with a verified account as a point of sales for business. The application uses Venmo's API to do the transaction. Yallpay.me utilizes face recogniztion (Face++ API) to authenticate user to their account; this allows user to make transaction without having to bring any credit/debit card or other money forms. In the future, this project can be expanded to utilize other forms of bioauthentication, such as fingerprint, to authorize transactions on the app.

Huddle

Huddle is an on-demand mobile application that connects students on campus. It is a safe environment for students to meet new people on campus based on their interests.
